🚀 SUITDEM ERP — Déploiement Clever Cloud

Guide testé et validé. Durée estimée : 20 minutes.

1 Prérequis locaux

Git installé → vérifier : git --version
Node.js installé → vérifier : node --version

2 Créer le compte Clever Cloud

  1. Aller sur clever-cloud.com → S'inscrire
  2. Accéder à console.clever-cloud.com

3 Ajouter la clé SSH sur GitHub ET Clever Cloud

Dans le Terminal :

ssh-keygen -t ed25519 -C "ton@email.com"   # Entrée x3
cat ~/.ssh/id_ed25519.pub                   # Copier le résultat

Ajouter cette clé sur deux endroits :

  • GitHub → Settings → SSH and GPG keys → New SSH key
  • Clever Cloud → Profil → SSH Keys → Add key
⚠️ Si la clé n'est pas sur GitHub, le push échouera avec "Permission denied".

4 Créer l'application PHP via GitHub

  1. Console → Create… → An application
  2. Choisir GitHub comme source → sélectionner ton repo
  3. Type : PHP — ⚠️ pas FS Bucket, pas Node.js
  4. Région : Paris, Plan : Nano, Nom : suitdem-erp
✅ Clever Cloud surveille GitHub et redéploie automatiquement à chaque push.

5 Créer l'add-on MySQL

  1. Console → Create… → An add-on → MySQL
  2. Plan : DEV (gratuit), Nom : suitdem-db
  3. Lier à l'app : suitdem-erp → Service dependencies → Edit → suitdem-db

6 Récupérer les credentials MySQL

Console → suitdem-db → Addon dashboard → Connection settings

ChampExemple→ config.php
Hostbxxxxx.mysql.clever-cloud.comDB_HOST
Port3306DB_PORT
DatabasebxxxxxDB_NAME
UsernameuxxxxxDB_USER
PasswordxxxxxxDB_PASS

7 Importer le schéma SQL

  1. Console → suitdem-db → Addon dashboard → phpMyAdmin
  2. Cliquer sur le nom de la base dans la colonne gauche pour la sélectionner
  3. Onglet Importer → choisir suitdem_schema.sql → Importer
  4. ✅ 9 tables créées
⚠️ Le fichier SQL ne doit pas contenir CREATE DATABASE ni USE — la base est déjà créée.

8 Modifier config.php

Dans suitdem_CC/api/config.php — remplacer les credentials MAMP :

define('DB_HOST', 'bxxxxx.mysql.clever-cloud.com');
define('DB_PORT', '3306');
define('DB_NAME', 'bxxxxx');
define('DB_USER', 'uxxxxx');
define('DB_PASS', 'votre_password');
define('API_TOKEN', 'suitdem_secret_2025');

9 Déployer via Git → GitHub → Clever Cloud

cd ~/Desktop/suitdem_CC
git init
git add .
git commit -m "SUITDEM ERP v19.75 Clever Cloud"
git remote add origin git@github.com:TON-USERNAME/suitdem.git
git push origin main --force

Suivre le déploiement : console → suitdem-erp → Activity → attendre ✅ vert (~2 min).

⚠️ Si "Everything up-to-date" mais pas de redéploiement : 
git commit --allow-empty -m "force redeploy"
git push origin main --force

10 Vérifier

  1. URL de l'app : console → suitdem-erp → Domain names
  2. Tester l'API : https://ton-app.cleverapps.io/api/get.php → doit retourner du JSON
  3. Dans l'app → Admin → bouton 🐘 actif → données chargées ✅

🔄 Mise à jour après la 1ère fois

cd ~/Desktop/suitdem_CC
git add .
git commit -m "description"
git push origin main

➡️ Migration sur LWS ensuite

  1. Modifier config.php → credentials LWS MySQL (port 3306)
  2. Importer suitdem_schema.sql dans phpMyAdmin LWS
  3. Uploader tout via FTP dans public_html/
✅ Même code, même schéma — seul config.php change.
