🚀 SUITDEM ERP — Déploiement Clever Cloud

Objectif : App en ligne sur Clever Cloud (PHP + MySQL) pour tester avant migration sur LWS. Un seul fichier à modifier.

Sommaire 1. Créer un compte Clever Cloud 2. Créer l'application PHP 3. Créer le add-on MySQL 4. Récupérer les credentials MySQL 5. Modifier config.php 6. Importer le schéma SQL 7. Déployer les fichiers (Git ou FTP) 8. Tester en ligne 9. Passage sur LWS ensuite

1 Créer un compte Clever Cloud

URL : clever-cloud.com → S'inscrire (gratuit, pas de CB requise pour le free tier)
  1. Créer un compte avec email ou GitHub
  2. Vérifier l'email
  3. Accéder à la console : console.clever-cloud.com

2 Créer l'application PHP

  1. Dans la console → Créer une application
  2. Choisir PHP
  3. Région : Paris (PAR) — serveurs français, plus proches de LWS
  4. Plan : Nano (gratuit pour les tests)
  5. Nom : suitdem-erp
  6. Cliquer Créer
⚠️ Ne pas encore déployer — on configure d'abord la base MySQL.

3 Créer le add-on MySQL

  1. Dans ton application → onglet Add-ons
  2. Cliquer Ajouter un add-on
  3. Choisir MySQL
  4. Plan : DEV (gratuit — 256MB, suffisant pour les tests)
  5. Nom : suitdem-db
  6. Cliquer Provisionner

4 Récupérer les credentials MySQL

  1. Dans la console → Add-ons → cliquer sur suitdem-db
  2. Onglet Informations de connexion
  3. Noter les 5 valeurs :
VariableExempleÀ copier dans config.php
Hostbxxxxx.mysql.clever-cloud.comDB_HOST
Port3306DB_PORT
DatabasebxxxxxDB_NAME
UsernameuxxxxxDB_USER
PasswordxxxxxxxxxxDB_PASS

5 Modifier config.php

Ouvrir api/config.php et remplacer le bloc MAMP par les valeurs Clever Cloud :

<?php
// ── Clever Cloud ──────────────────────────────────────────
define('DB_HOST', 'bxxxxx.mysql.clever-cloud.com');
define('DB_PORT', '3306');
define('DB_NAME', 'bxxxxx');
define('DB_USER', 'uxxxxx');
define('DB_PASS', 'votre_password');

define('API_TOKEN', 'suitdem_secret_2025');  // ne pas changer

define('ALLOWED_TABLES', ['clients','products','insurance','services',
                          'options','misc','vehicles','team','operations']);
// ... reste du fichier inchangé
✅ C'est le seul fichier à modifier. Tout le reste est identique à MAMP.

6 Importer le schéma SQL

Clever Cloud fournit un accès phpMyAdmin pour ton add-on MySQL :

  1. Dans la console → Add-on suitdem-db → bouton phpMyAdmin (ou URL fournie)
  2. Onglet Importer
  3. Choisir le fichier suitdem_schema.sql
  4. Cliquer Importer
  5. ✅ 9 tables créées
⚠️ Si phpMyAdmin n'est pas disponible directement, utiliser un client MySQL desktop comme TablePlus (gratuit) ou Sequel Pro (Mac, gratuit) avec les credentials de l'étape 4.

7 Déployer les fichiers

Clever Cloud supporte deux méthodes — choisis la plus simple :

Option A — Git (recommandé)

  1. Dans la console → ton app → onglet Informations → copier l'URL Git
  2. Dans ton dossier suitdem/ :
git init
git add .
git commit -m "SUITDEM ERP v19.75 initial deploy"
git remote add clever <url-git-clever-cloud>
git push clever master

Clever Cloud détecte PHP automatiquement et déploie. L'app est live en ~2 minutes.

Option B — FTP/SFTP

  1. Dans la console → ton app → onglet FTP → activer l'accès FTP
  2. Utiliser FileZilla avec les credentials FTP fournis
  3. Uploader tout le contenu de suitdem/ dans le dossier public/ (ou htdocs/ selon la config)
⚠️ Le dossier racine de l'app PHP sur Clever Cloud est public/ (pas public_html/ comme LWS). Uploader directement le contenu dedans, pas le dossier lui-même.

Structure à uploader

    ├── get.php
    ├── save_operation.php
    ├── save_client.php

    └── delete_row.php

8 Tester en ligne

  1. URL de l'app : fournie par Clever Cloud après déploiement, format https://suitdem-erp.cleverapps.io
  2. Ouvrir l'app → Panneau Admin → vérifier que le bouton 🐘 PHP est actif
  3. Tester : créer un client, créer une opération, sauvegarder
  4. Vérifier dans phpMyAdmin Clever Cloud que les données apparaissent
✅ Si tout fonctionne ici, le déploiement sur LWS sera identique — juste changer config.php à nouveau.
